# CV-App 프로젝트

## 프로젝트 개요
- OpenCV 기반 컴퓨터 비전 포트폴리오 프로젝트
- 목표: 3월까지 면접용 포트폴리오 완성
- 사용자: 비개발자, Python/Git 기초 지식 있음, OpenCV 기본 지식 있음

## 기술 스택
- Python 3.12 (uv로 관리)
- OpenCV, NumPy, Pillow, Matplotlib
- Git 버전관리
- uv 패키지 매니저 (경로: C:\Users\kim joonsik\.local\bin\uv.exe)

## 프로젝트 구조
```
cv-app/
├── src/          # 소스 코드
├── data/images/  # 이미지 데이터 (Git 미포함)
├── outputs/      # 결과물 (Git 미포함)
├── notebooks/    # Jupyter 노트북
├── main.py       # 메인 코드
└── pyproject.toml
```
